# Changelog

## 0.2.0 - 2026-03-30

- Added a multi-species aquarium ecosystem with sharks, jellyfish, eels, reef fish, turtles, octopus, swordfish, and starfish.
- Added a shared simulation core that powers both offline local mode and authoritative WebSocket multiplayer rooms.
- Added hosting/join controls, shared cursor sync, chat bubbles, join/leave toasts, and LAN room codes.
- Added macOS packaging config for `electron-builder`, native menu support, dock badge wiring, and fullscreen-ready window behavior.
